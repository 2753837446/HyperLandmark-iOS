# HyperLandmark-iOS

A 68-point Facial landmark at 28-30 FPS on iPhone 6S+. HyperLandmark is get from https://github.com/zeusees/HyperLandmark.

## Resources
- [SDM](https://github.com/chengzhengxin/sdm)
